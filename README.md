# Demoth

T-shirt design studio built with Next.js 16. Design your own clothes.

Live: https://demoth.netlify.app

## Drawing recognition

When the **Auto-correct** premium toggle is on, finishing a freehand
drawing surfaces "Did you draw one of these?" suggestions for any of 14
built-in shapes (circle, square, triangle, diamond, pentagon, hexagon,
star, heart, moon, droplet, lightning, leaf, arrow, cross). Pick one and
the wobbly drawing is replaced with a clean SVG.

Recognition runs entirely client-side — no API keys, no network, no cost.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
